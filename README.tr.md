# Portfolio Website

**Diller:** [🇺🇸 English](README.md) | [🇹🇷 Türkçe](README.tr.md)

Temiz ve modern kişisel portfolio websitem, web geliştirme projelerimi ve becerilerimi sergiliyor.

## 🌐 Canlı Demo

Canlı siteyi ziyaret edin: [tolgaboz.me](https://tolgaboz.me)

## ✨ Özellikler

- **Duyarlı Tasarım**: Tüm ekran boyutları ve cihazlar için optimize edilmiş
- **Modern Arayüz**: Temiz, minimalist tasarım ve akıcı animasyonlar
- **Proje Vitrini**: Öne çıkan projeleri önizlemeleriyle gösteren grid düzeni
- **Hakkımda Bölümü**: Kişisel bilgiler ve geçmiş
- **İletişim Formu**: Kolay iletişim kurma imkanı
- **CV Erişimi**: İndirilebilir CV'ye doğrudan link

## 🛠️ Kullanılan Teknolojiler

- **HTML5** - Semantik işaretleme
- **CSS3** - Flexbox/Grid ile modern stillendirme
- **JavaScript** - Etkileşimli işlevsellik
- **Google Fonts** - Tipografi (Montserrat, Open Sans, Josefin Sans)

## 📁 Proje Yapısı

```
portfolio/
├── index.html          # Ana sayfa
├── style.css           # Genel stiller
├── about/
│   └── index.html      # Hakkımda sayfası
├── contact/
│   └── index.html      # İletişim sayfası
├── assets/
│   ├── favicon.png     # Site ikonu
│   ├── resume.pdf      # İndirilebilir CV
│   └── previews/       # Proje önizleme görselleri
└── CNAME              # Özel domain yapılandırması
```

## 📱 Duyarlı Tasarım

Website tamamen duyarlı ve şu cihazlarda test edilmiştir:

- Masaüstü (1920px+)
- Laptop (1024px - 1919px)
- Tablet (768px - 1023px)
- Mobil (320px - 767px)

## 🔧 Özelleştirme

Bu portfolio'yu kendi kullanımınız için özelleştirmek için:

1. `index.html` dosyasındaki kişisel bilgileri güncelleyin
2. Proje linklerini ve açıklamalarını değiştirin
3. `assets/previews/` klasörüne kendi proje önizleme görsellerinizi ekleyin
4. `assets/resume.pdf` dosyasını kendi CV'nizle güncelleyin
5. Tercihlerinize göre `style.css` dosyasındaki stilleri değiştirin
